_Title_
Repository Importer - LLScript



_Short description_
Importieren Sie Scripts mit nur einem Klick in Lightning Launcher



_Complete description_
*********
Lightning Launcher muss hierfür installiert sein.
https://play.google.com/store/apps/details?id=net.pierrox.lightning_launcher_extreme

Alle Texte in der App sind auf Englisch
*********

Diese App besteht aus einem Browser mit dem Sie das Script Repository im LL-Wiki durchsuchen können. Wenn Sie ein Script gefunden haben, drücken Sie einfach den Downloadbutton und importieren es (oder teilen es alternativ als Text)

Diese App wurde erstellt von Lukas Morawietz und TrianguloY.

Google+ Seite vom Lightning Launcher
https://plus.google.com/communities/109017480579703391739

Sie finden den Sourcecode hier:
https://github.com/TrianguloY/LLScript_Repository_Importer
